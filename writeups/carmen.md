# CARMEN

- Catégorie : Programmation / Puzzle
- Difficulté : Non précisée
- Flag : `SHLK{carmen_rings_restored}`

## Démarche

Challenge interactif limité à 60 secondes, avec 2 observations libres et 20 rotations maximum. Les commandes manipulaient des anneaux : A+3, B-1, OBSERVE A, REFERENCE, TARGET, EXIT. L'objectif était d'aligner les marqueurs cibles sur les anneaux de référence en minimisant observations et rotations.

## Commandes et scripts pertinents

Aucun script public associé.

```bash
Commandes interactives : OBSERVE, REFERENCE, TARGET, rotations A/B/... puis EXIT.
```

## Conclusion

Le flag exact est publié, mais la séquence finale de rotations n'est pas reprise ici.
